# WeatherApp (An Assessment submitted to EY)
WeatherApp is an application that gets the weather forecast of some top cities. This include:
* Kenya
* Cairo
* Texas
* Amazon
* Peru etc.

# Technologies Used
1. Kotlin
2. Kotlin Flow
3. Coroutines
4. Retrofit
5. LiveData
6. MVVM Architecture
7. Navigation Component
8. DaggerHilt
9. DataBinding
10. Coil
11. DiffUtill
